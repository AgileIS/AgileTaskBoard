Agile Task Board release notes

Version 3.2.0.0​

New Features

  • Adding images to the task description is now possible
  • Integrated the rich text editor from the Teams version into the SPFx version
    • The use of the old (legacy) rich text editor can be configured in the settings
  • (Teams/Group) members can be utilized to be displayed in the header bar
    • Can be enabled inside the settings panel Use (Team/Group) Members

Changes

  • No admin consent required for Agile Task Board in Teams!
    • Certain features still require admin consent, which can be granted via the Settings Panel/Permissions.

Bug Fixes

  • Minor bug fixes

Version 3.1.0.0​

New Features

  • Live updates (allows to receive changes in near real time)
  • Added the possibility to resolve version conflicts of task elements per field inside the edit panel
  • Added export and import of Agile Task Board instance settings to the settings panel

Changes

  • Improved performance
  • Updated Task Edit Panel UI
  • Added option to hide the sub heading property on the task card

Version 3.0.0.0​

New Features

  • Planner integration
  • Notifications (Teams Activity Feed) for mentioned in comments and task assign to [Teams only]
  • Moved to modern lists (generic list type)
  • Use list item comments

Changes

  • Removed support for SharePoint on premise (2016,2019)
  • Improved start trial

Bug Fixes

  • Several minor UI/UX fixes

Version 2.2.5.0​

New Features

  • Document Preview inside teams app

Changes

  • Updated error handling and logging.
  • Add document to task on upload
  • Added environment information inside the settings panel
  • Added more action to task (mobile view)

Bug Fixes

  • Several minor UX bug fixes.
  • Missing document library inside teams app
  • Minor bug fixes

Known Issues

  • Touch DnD partial support for ie11 and Edge
  • IE11 input cursor position on task (card) inputs only changeable with keyboard

Version 2.2.4.0​

New Features

  • We have redesigned the Teams app so that it now runs natively in teams and interacts with the Microsoft Graph.
  • Teams deep link support
  • The SharePoint Online App and the Teams app will soon be published in the Office Store and can be retrieved from there.

Changes

  • SPFx "Sync to Teams" function is no longer available due to the new native Teams App.

Bug Fixes

  • Update user properties after changing them in the profile, like name and image.
  • Several minor UX bug fixes.

Known Issues

  • Touch DnD partial support for ie11 and Edge
  • IE11 input cursor position on task (card) inputs only changeable with keyboard

Version 2.2.3.0​

New Features

  • Wizard Templating mechanism to generate pre-configured boards
    • Empty Template: Set up your own board by starting with an empty template.
    • Kanban - Simple Board: The simple Kanban board template is an easy starting point for agile planning methods.
    • Scrum - Sprint Board: Use the scrum sprint board together with the scrum planning board for lightweight scrum processes.
    • Scrum - Planning Board: Use the scrum planning board together with the scrum sprint board for lightweight scrum processes.
  • Status field choices editable inside the settings panel
  • Preferred Tags changeable within the mobile view

Changes

  • Redesign of the document upload
    • the UI is now closer to the SharePoint (online) styles
    • removed categories (documents subdivision): "Documents / Documents to Upload / Documents to Attach"
    • uploading documents now on drop / select (upload button), previously the upload was triggered on save button click
  • Removed "SupportedLocales" from AppManifest, because of the error: "The app package used for upgrade does not support the culture 'xx-XX' used for installation."
    • The WebPart still supports the languages English and German

Bug Fixes

  • Minor bug fixes

Known Issues

  • Touch DnD partial support for ie11 and Edge
  • IE11 input cursor position on task (card) inputs only changeable with keyboard

Bug Fixes

  • Some UI and theming fixes

Known Issues

  • Touch DnD partial support for ie11 and Edge
  • IE11 input cursor position on task (card) inputs

Update info SharePoint 2019

if the agiletaskboard-sp2016.sppkg is installed in the App Catalog please follow the steps below:

  • Select the agiletaskboard-sp2016.sppkg list entry
  • Use the Ribbon or Item Menu to select Rename or Edit Properties
  • Change the Name property from agiletaskboard-sp2016 to agiletaskboard-sp2019
  • Upload and Deploy the agiletaskboard-sp2019.sppkg file
